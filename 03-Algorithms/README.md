# Algorithms

## Overview

This module explores classical algorithms and optimization techniques used throughout software engineering. Emphasis is placed on correctness, efficiency, and complexity analysis.

---

## Concepts Covered

- Searching Algorithms
- Sorting Algorithms
- Divide & Conquer
- Greedy Techniques
- Time Complexity
- Space Complexity

---

## Objectives

- Understand algorithm design principles.
- Compare multiple algorithmic approaches.
- Analyze computational complexity.
- Build efficient and scalable solutions.
